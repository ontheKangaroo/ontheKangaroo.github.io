# Finite, Countable, and Uncontable Sets

Definition 3

If there exists a 1-1 mapping of A onto B, we say that A and B, can be put in 1-1 correspondense, or that A and B have the same cardinal number, or briefly, that A and B are equivalent, and we write A∼BA \sim BA∼B. This relation clearly has the following properties:

  1. It is reflexive: A∼AA \sim AA∼A.
  2. It is symmetric: If A∼BA \sim BA∼B, then B∼AB \sim AB∼A.
  3. It is transitive: If A∼BA \sim BA∼B and B∼CB \sim CB∼C, then A∼CA \sim CA∼C.

Any relation with these three properties is called an equivalence relation.
