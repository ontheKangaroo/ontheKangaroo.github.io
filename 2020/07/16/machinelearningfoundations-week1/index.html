# When to use machine learning

  • learning: observations -> learning -> skill

  • machine learning: data -> ML -> skill

  • skill: improve some performance measure / prediction accuracy

# Why to use machine learning

  • ML: an alternative route to build complicated systems (other than hand-program)

    • cannot program the system manually

    • cannot define the solution easily

    • rapid decisions

    • user-oriented in a massive scale

  • ML: key essence

    • exists some underlying pattern to be learned

    • no programmable definition

    • data about the pattern
