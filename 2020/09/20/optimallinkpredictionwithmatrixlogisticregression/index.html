# Optimal Link Prediction with Matrix Logistic Regression

by Nicolai Baldin and Quentin Berthet

# 0. Introduction

  • link prediction: predicting the presence or absence of edges in a large graph, based on the observations of some of its edges, and on side information

  • unsupervised statistical problems based on graphs: the graph itself is the sole data, there is no side information, and the objective is to recover an unknown structure in the generative model

  • supervised statistical problem based on graphs: understanding a fundamental mechanism, formalized as the link between two variables. The objective is to learn how an explanatory variable XXX allows to predict a response YYY

  • application

    • social network

    • citation network / interaction between molecules or proteins
