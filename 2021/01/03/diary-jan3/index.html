# Introduction

A gambler must choose which of KKK slot machines to play. At each time step, he pulls the arm of one of the machines and receives a reward or payoff. The gambler's purpose is to maximize his return. In this model, each arm is assumed to deliver rewards that are independently drawn from a fixed and unkown distribution. As reward distributions differ from arm to arm, the goal is to find the arm with the highest expected payoff as early as possible and then to keep gambling using that best arm.

The gambler's performance is typically measured in terms of "regret". This is the differene between the expected return of the optimal strategy and the gambler's expected return.

Adversarial bandit problem: no statistical assumptions are made about the generation of rewards. Assume only that each slot machine is initially assigned an arbitrary adn unknown sequence of rewards, one for each time step, chosen from a bounded real interval.

Given any sequence (j1,...,jT)(j_1, ..., j_T)(j​1​​,...,j​T​​) of pulls, where T>0T > 0T>0 is an arbitrary time horizon and each jtj_tj​t​​ is the index of an arm, the wordt-case regret of a gambler for this sequence of pulls is the difference between the return the gambler would have had by pulling arms j1,...,jTj_1, ..., j_Tj​1​​,...,j​T​​ and the actual gambler's return, where both returns are determined by the initial assignment of rewards.

Allow the regret to depend on the "hardness" of the sequence of pulls for which it is measured, where the hardness of a sequence is roughly the number of times one has to change the slot machine currently being played in order to pull the arms in the order given by the sequence.
